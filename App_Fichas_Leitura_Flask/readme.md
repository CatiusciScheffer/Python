

# List Book App

List Book App é uma aplicação web desenvolvida com Flask que permite gerenciar uma lista de livros. Os usuários podem adicionar, editar e excluir registros de livros, além de visualizar todos os livros adicionados.

## Funcionalidades

* **Adicionar Livro** : Permite ao usuário adicionar um novo livro preenchendo um formulário com informações detalhadas.
* **Editar Livro** : Permite ao usuário editar os detalhes de um livro existente.
* **Excluir Livro** : Permite ao usuário excluir um livro da lista.
* **Listar Livros** : Exibe uma lista de todos os livros adicionados, com suas informações.

## Tecnologias Utilizadas

* **Python** : Linguagem principal utilizada para desenvolver a aplicação.
* **Flask** : Framework web utilizado para criar a aplicação.
* **SQLite** : Banco de dados utilizado para armazenar as informações dos livros.
* **Flask-SQLAlchemy** : Extensão do Flask para integrar SQLAlchemy, um ORM para o banco de dados.
* **webview** : Biblioteca utilizada para criar uma janela de aplicativo de desktop a partir do aplicativo web.

### Funcionalidades

#### Adicionar Livro

1. Navegue até `http://127.0.0.1:5000/add`.
2. Preencha o formulário com as informações do livro.
3. Clique em "Salvar" para adicionar o livro.

#### Editar Livro

1. Na página principal (`http://127.0.0.1:5000/`), clique em "Editar" ao lado do livro que deseja editar.
2. Atualize as informações no formulário.
3. Clique em "Salvar" para salvar as alterações.

#### Excluir Livro

1. Na página principal (`http://127.0.0.1:5000/`), clique em "Excluir" ao lado do livro que deseja excluir.
