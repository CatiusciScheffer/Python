# Aplicativo ToDo

Este é um aplicativo simples de ToDo construído usando Flet e SQLite para gerenciar tarefas. O aplicativo permite adicionar, visualizar e excluir tarefas, além de marcá-las como concluídas.

## Funcionalidades

* Adicionar novas tarefas
* Visualizar todas as tarefas
* Marcar tarefas como concluídas
* Excluir tarefas individuais
* Excluir todas as tarefas
* Visualizar tarefas por status (Todas, Em Andamento, Concluídas)
